swishflo

The swishflo effect adds a nice clean touch to any button. Use on blog tags, block links, navbar links, and so forth. Simply copy and paste the following code

<script src="https://mikjakrych.github.io/swishflo/swishflo.js" charset="utf-8"></script>

into your document and use the class name swishflo on any object that you want to apply swishflo to.

Here are a few nice swishflo features:

  • swishflo can be applied to any element not just <button> elements
  • swishflo has been tested thoroughly and works on Chrome, Edge, IE, and Firefox
  • swishflo delays the button's action so that users can enjoy the feature before moving on.
  • swishflo can be used anywhere free of charge
